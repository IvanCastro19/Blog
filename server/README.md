## Descripción del servicio
CRUD para Blog web

## Comandos
### Instalación de dependencias
- Para instalar todas las dependencias.

        npm install || npm i
    
- Si se quiere excluir las dependencias de desarrollo (librerías para testing).
    
        npm install --omit=dev || npm i --omit=dev
    
### Iniciar el servidor

        npm start
